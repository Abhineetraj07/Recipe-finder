
# 🍽️ Recipe Finder

The Recipe Finder is a web application built using **HTML, CSS, and JavaScript** with Firebase and Spoonacular API integration. It allows users to quickly search for recipes, view detailed instructions, and access YouTube videos if no recipe data is available. The app supports **user login**, **favorites management**, and a **clean, responsive UI** for an easy cooking experience.  

---

## 🚀 Features

### 🔍 Recipe Search
Search recipes by keywords with real-time API results.

### 📹 YouTube Fallback
If no recipe data is found, it fetches YouTube video tutorials automatically.

### 💾 Favorites Management
Save favorite recipes using Firebase for easy access later.

### 👤 User Authentication
Login/signup functionality using Firebase Authentication.

### 🌓 Dark Mode Support
Toggle between light and dark themes for better accessibility.

### 📱 Responsive Design
Works seamlessly on desktops, tablets, and mobile devices.

## ⚙️ Setup Instructions

### Clone the Repository:
```bash
git clone https://github.com/Abhineetraj07/Recipe-finder.git
```

### Configure Firebase:
1. Create a project in [Firebase Console](https://console.firebase.google.com/)  
2. Enable **Authentication** and **Firestore Database**  
3. Add your `firebaseConfig` credentials in the JavaScript file  

### Add API Keys:
- Get Spoonacular API Key from [Spoonacular](https://spoonacular.com/food-api)  
- Get YouTube API Key from [Google Cloud Console](https://console.cloud.google.com/)  

### Run Locally:
Open the `index.html` file in your browser or use a simple local server.
